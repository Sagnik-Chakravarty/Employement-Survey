# Survey Instrument

This directory is intended to store the original Qualtrics `.qsf` export for the Current Employment Survey instrument.

Recommended file name:

```text
current_employment_survey.qsf
```

The `.qsf` file can be imported directly into Qualtrics using **Create Project → From a File → Import Survey**.

The documentation in `docs/` was generated from the Qualtrics export and summarizes the instrument structure, question wording, and survey-design purpose.
