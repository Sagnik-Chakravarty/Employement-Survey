# Figures

This directory contains visual documentation for the Current Employment Survey instrument.

## Files

- `survey_flow_diagram.md`: Markdown/Mermaid version of the survey flow diagram.
- `survey_flow_diagram.mmd`: Raw Mermaid source for the survey flow.
- `survey_flow_diagram.svg`: SVG figure showing the main questionnaire flow from consent through household roster, person-level employment questions, hours/earnings, and household income.
- `measurement_domains.svg`: SVG figure summarizing the main measurement domains: household roster, demographics, employment status, hours worked, and earnings/income.
- `qa_workflow.svg`: SVG figure summarizing the survey QA and response-data workflow.

## Use

These figures are intended for portfolio review, survey documentation, and explaining the instrument structure to technical and nontechnical audiences.
